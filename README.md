# PointMap

## How to use

Install it and run:

```sh
# clone the project
git clone https://github.com/dedzec/PointMap.git

# enter the project directory
cd PointMap

# install dependency
npm install

# For Android run:
react-native run-android

# For iOS run:
react-native run-ios

# For run project:
react-native start
```
